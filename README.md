# 147 дней в тундре - Дневник Фёдора Овчинникова

Этот сайт представляет собой удобную версию для чтения дневниковых записей Фёдора Овчинникова из его путешествия в тундру, отсортированных в хронологическом порядке.

## ⚠️ Важно

Этот сайт является копией оригинального сайта Фёдора Овчинникова: https://tundra147.com/

Все права на контент принадлежат Фёдору Овчинникову.

## Структура проекта

- `index.html` - главная страница с дневниковыми записями
- `style.css` - стили в стиле тундры с адаптивным дизайном
- `script.js` - автоматическая сортировка записей по датам
- `hero-image.jpg` - главное изображение сайта

## Особенности

- ✅ Все 369 записей из оригинального дневника
- ✅ Записи отсортированы в хронологическом порядке (от 20 марта до 12 августа 2025)
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Удобные для чтения шрифты (Cormorant Garamond для заголовков, Inter для текста)
- ✅ Плавные анимации при загрузке
