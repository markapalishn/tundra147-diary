# 147 дней в тундре - Дневник Фёдора Овчинникова

Этот сайт представляет собой удобную версию для чтения дневниковых записей Фёдора Овчинникова из его путешествия в тундру, отсортированных в хронологическом порядке.

## ⚠️ Важно

Этот сайт является копией оригинального сайта Фёдора Овчинникова: https://tundra147.com/

Все права на контент принадлежат Фёдору Овчинникову.

## Структура проекта

- `index.html` - главная страница с дневниковыми записями
- `style.css` - стили в стиле тундры с адаптивным дизайном
- `script.js` - автоматическая сортировка записей по датам и календарь навигации
- `Tundra 147.mov` - видео на главной странице

## Особенности

- ✅ Все 369 записей из оригинального дневника
- ✅ Записи отсортированы в хронологическом порядке (от 20 марта до 12 августа 2025)
- ✅ Календарь навигации с переключением между "Дни" и "Даты"
- ✅ Счетчик дней в тундре для каждой записи (начиная с 23 марта 2025)
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Шрифт Inter для всего сайта
- ✅ Упрощенная цветовая палитра (белый, черный, серый, акцентный синий)

## Развертывание на GitHub Pages

1. Создайте новый репозиторий на GitHub (например, `tundra147-diary`)

2. Выполните скрипт развертывания:
```bash
./deploy.sh <имя-репозитория> <ваш-github-username>
```

3. В настройках репозитория GitHub (Settings > Pages):
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
   - Сохраните настройки

4. Ваш сайт будет доступен по адресу: `https://<ваш-username>.github.io/<имя-репозитория>/`
